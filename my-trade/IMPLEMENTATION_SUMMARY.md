## 🚀 快速使用

### 方式1: 查询单个基金

```python
from fund_api import FundAPI

api = FundAPI()

# 查询华夏成长混合
fund_info = api.get_fund_realtime_value("000001")
if fund_info:
    api.print_fund_info(fund_info)
```

### 方式2: 获取投资建议（含AI分析）

```python
from fund_api import FundAPI
from fund_investment_advisor import FundInvestmentAdvisor

# 初始化
api = FundAPI()
advisor = FundInvestmentAdvisor(use_llm=True)

# 查询易方达黄金主题
fund_info = api.get_fund_realtime_value("161116")

if fund_info:
    # 分析并生成AI建议
    analysis = advisor.analyze_fund(fund_info)
    advisor.print_advice(analysis)
```

## 💡 投资原则（来自 docs/skills.md）

系统已集成以下核心投资原则：

1. **永远不要追涨杀跌** ❌ 上涨时买入 ✅ 下跌时买入
2. **不幻想买在最低卖在最高** - 在合理区间操作即可
3. **沉没成本不参与决策** - 方向错了要及时止损
4. **红的才卖，绿的套着** - 盈利止盈，亏损持有（除非趋势反转）
5. **关注贵金属特性** - 金银、铜、铝等不同商品的特点

## 🎓 适用场景

### ✅ 适合
- 场外基金（开放式基金、QDII等）
- 日常监控和决策辅助
- 寻找买入/卖出机会

### ❌ 不适合
- 场内交易基金（ETF）- 需使用其他API
- 历史回测 - 目前仅支持实时数据
- 高频交易 - 数据更新有延迟

## ⚠️ 重要提示

1. **数据时效**: 实时估值仅在交易日更新，非交易时间可能显示"---"
2. **仅供参考**: 所有建议仅供参考，不构成投资建议
3. **风险自负**: 基金投资有风险，入市需谨慎
4. **及时决策**: 建议每日盘前或盘后运行监控工具
